## Tech Stack

- **React**
- **Tailwind CSS**

## How to use

1. Clone the repo:

   ```
   git clone https://github.com/salmanahmed-chd/SavorSpot.git
   ```

2. Install dependencies:

   ```
   npm install
   ```

3. Start the dev server:
   ```
   npm run dev
   ```
